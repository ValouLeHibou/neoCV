Hackathon - Projet Madu

Dans le cadre d'un hackathon organisé par l'ECV Digital, mon équipe et moi-même avons eu à développer une application mélangeant Tripadvisor et Google Maps. Étant le seul développeur de l'équipe j'ai pu réaliser le projet avec les technologies que je souhaitais : à savoir Python, Docker ou encore Postgres. Sur les 3 jours de hackathon, j'ai eu le temps de réaliser une interface Mapbox (un Maps open source) installé dans une infrastructure composée de technologies de cartographie comme Umap, Qgis ou encore PostGis. Ainsi, grâce à nos efforts, peu de sommeil et beaucoup de café, nous avons remporté le hackathon.
